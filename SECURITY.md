# Security Policy

## Supported Versions

Laravel Alias Manager is currently in early development and has no stable release yet.

## Reporting a Vulnerability

Please do not report security vulnerabilities through public GitHub issues.

If you discover a security issue, contact the maintainer privately.

Include:

- A clear description of the issue
- Steps to reproduce
- Potential impact
- Suggested fix, if available

We will review valid reports and publish fixes as appropriate.
